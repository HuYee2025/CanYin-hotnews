# CanYin-hotnews 餐饮品牌经营热点搜索

基于张总餐饮品牌顾问视角，为“张总短视频”项目搜索、筛选和核查餐饮行业新闻。

这个技能不是泛餐饮热榜工具。它优先寻找能转化为短视频选题的经营事件：老板痛点、反常识判断、案例证据和可执行动作。

激活关键词：餐饮热点、餐饮新闻、今天适合张总讲的新闻、餐饮品牌案例、餐饮行业选题。

## v2.1.1 更新重点

- 统一技能名称为 `CanYin-hotnews`，与 GitHub 仓库名和本地安装目录保持一致。

## v2.1.0 更新重点

本次更新把搜索逻辑从“抓热榜 + 关键词过滤”升级为“经营议题优先 + 多源核查 + 短视频转化评分”。

- 搜索主线：行业经营资讯、品牌案例追踪、平台热榜扫描、政策食安与原料波动。
- 筛选规则：一条内容必须同时命中餐饮场景词和经营维度词，避免“吃、菜、烤、打卡”等泛词误抓。
- 可靠性分级：T1 官方/公告/监管，T2 行业和财经媒体，T3 热榜和自媒体线索。
- 核查规则：门店数、营收、客单价、处罚、关店、融资等事实必须回溯来源；热榜不能单独作为事实依据。
- 转化评分：按经营相关性、证据可靠性、时效性、短视频转化性打分，8 分以上进入“张总精选”。
- 输出升级：新增来源等级、可靠性标签、事实核查、不出稿清单和选题转化卡。

## 适合搜索什么

优先搜索这些内容：

- 餐饮品牌开店、关店、扩张、加盟、收缩、财报、融资、并购。
- 门店模型、招牌菜、菜单结构、定价、复购体验、标准化复制。
- 区域餐饮品牌、地方菜、城市菜、老字号年轻化。
- 头部品牌可借鉴打法，例如海底捞、西贝、九毛九/太二、巴奴、费大厨、旺顺阁、烤匠、老乡鸡、塔斯汀、米村拌饭等。
- 食安、供应链、原料波动、监管政策。

默认排除这些内容：

- 饮料快消、零食、预包装食品。
- 咖啡茶饮新品、联名、代言、娱乐化热搜。
- 探店八卦、吃播段子、网友吐槽、单纯“好吃/打卡/排队”话题。

## 输出格式

技能会输出：

1. 今日餐饮品牌经营热点一句话总判断。
2. 张总精选 3-5 条，每条包含来源等级、可靠性标签、核心判断、可拍角度、老板动作和事实核查。
3. 行业动态，只收录有参考价值但不适合深度展开的内容。
4. 不出稿清单，说明哪些热门内容被排除以及原因。

## 快速开始

```bash
# 微博热搜
curl -s "https://60s.viki.moe/v2/weibo" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value)热度)"'

# 知乎热榜
curl -s "https://60s.viki.moe/v2/zhihu" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value_desc))"'

# 抖音热点
curl -s "https://60s.viki.moe/v2/douyin" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value)热度)"'
```

热榜只作为线索。命中后还要搜索原始新闻、品牌公告、行业媒体或监管信息。

## 安装

OpenClaw:

```bash
openclaw skills install CanYin-hotnews
```

手动安装:

```bash
mkdir -p ~/.openclaw/workspace/skills/CanYin-hotnews
# 下载 SKILL.md 到该目录
```
