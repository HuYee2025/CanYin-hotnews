# china-hotnews: 餐饮热点洞察

从社会热点中挖掘餐饮营销切入点，为餐饮咨询顾问提供决策支持。

**适用于：OpenClaw / Claude Code / Codex / 其他AI助手**

## 触发关键词

**用户说"餐饮热点"时触发**

就这么简单！

## 支持平台

### 社交平台热点（60s API）

| 平台 | 端点 | 数据 |
|------|------|------|
| 微博热搜 | `/weibo` | 标题+热度值+链接 |
| 知乎热榜 | `/zhihu` | 标题+详情+热度+回答数 |
| 抖音热点 | `/douyin` | 标题+热度值+封面 |
| B站热门 | `/bili` | 标题+播放量 |
| 小红书热榜 | `/rednote` | 标题+热度+链接 |
| 百度热搜 | `/baidu/hot` | 标题+热度+描述 |

### 餐饮专业资讯（多源整合）

| 来源 | URL | 内容 | 语言 |
|------|-----|------|------|
| 红餐网热文 | https://m.canyin88.com/hot/ | 餐饮热点文章排行 | 中文 |
| 红餐网快讯 | https://www.canyin88.com/kuaixun/ | 餐饮行业快讯 | 中文 |
| 中国食品安全网 | https://www.cfsn.cn | 官方权威政策、农产品价格 | 中文 |
| Food Dive | https://www.fooddive.com | 全球餐饮产业深度新闻 | 英文 |

**AI筛选流程：**
1. 抓取多源餐饮资讯
2. AI分析每条资讯的餐饮营销价值
3. 筛选出最有价值的内容
4. 去重、去低质量内容
5. **英文内容翻译成中文**
6. 按营销价值排序输出

## API配置

**社交平台热点API（默认，免费无需Key）：**
```
https://60s.viki.moe/v2/{platform}
```

**餐饮专业资讯（红餐网）：**
- 热文频道：https://m.canyin88.com/hot/
- 快讯频道：https://www.canyin88.com/kuaixun/
- 使用web_fetch工具抓取文章内容

所有人安装后直接可用，无需额外配置。

**自建API（可选，长期稳定）：**

如果你想更稳定、不受公共API限制，可以自己部署：

```bash
# Docker部署
docker run -d --restart always --name 60s -p 4399:4399 vikiboss/60s:latest

# 或Node.js部署
git clone https://github.com/vikiboss/60s.git
cd 60s && pnpm install && PORT=4399 pnpm start
```

自建后修改配置文件：`~/.config/china-hotnews/config.json`
```json
{
  "api_base": "http://localhost:4399/v2",
  "default_platforms": ["weibo", "zhihu", "douyin"],
  "top_n": 5
}
```

**注意：** 自建API只有你自己能用，其他人默认使用公共API。

## 使用方式

### 1. 查看热搜（手动触发）

用户说"抓热点"、"看热搜"、"今日新闻"时执行：

```bash
# 默认使用公共API（所有人可用）
curl -s "https://60s.viki.moe/v2/weibo" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value)热度)"'

curl -s "https://60s.viki.moe/v2/zhihu" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value_desc))"'

curl -s "https://60s.viki.moe/v2/douyin" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value)热度)"'
```

```bash
API_BASE=$(cat ~/.config/china-hotnews/config.json 2>/dev/null | jq -r '.api_base' || echo "https://60s.viki.moe/v2")
curl -s "$API_BASE/weibo" | jq -r '.data[:5] | .[] | "• \(.title) (\(.hot_value)热度)"'

### 4. 餐饮热点洞察（餐饮咨询顾问专用）

用户说"餐饮热点"时触发：

**功能：从社会热点和餐饮专业资讯中挖掘餐饮营销切入点**

**人设：**
你是一个专业的餐饮新闻情报助手，擅长从社会热点中挖掘餐饮营销切入点，为餐饮咨询顾问提供决策支持。

**任务：**
1. 抓取今日社会热点新闻（微博/知乎/抖音/百度等）
2. 抓取红餐网餐饮专业资讯（热文/快讯）
3. 抓取中国食品安全网、Food Dive等数据源
4. AI分析每个热点是否有餐饮关联
5. 筛选出10条最有营销价值的热点
6. 提取餐饮切入点并给出营销建议

**筛选标准：**
- 当天热度高、全网关注度高
- 有餐饮营销价值
- 排除纯娱乐八卦
- **排除药品、医疗类话题**（如红霉素、感冒药等）

**排除关键词：**
- 药品：红霉素、感冒药、抗生素、止痛药、消炎药等
- 医疗：医院、医生、手术、治疗等

**输出格式：**

🍽️ 今日（X月X日）餐饮热点洞察

✍️ 总结：（100字内，简短精炼，概括今日餐饮营销机会）

---

🧩 餐饮切入点列表（按营销价值排序，共10条）

**01. [新闻标题](链接)**

新闻摘要：（100-150字，讲清楚事件核心+有意思的细节）

**餐饮切入点：** 具体切入点说明

**营销建议：** 具体可执行的营销建议

---

**02. [新闻标题](链接)**

新闻摘要：...

**餐饮切入点：** ...

**营销建议：** ...

---

...（共10条）

**格式要求：**
- 每条新闻用卡片式设计（标题+分隔线+内容）
- 编号使用粗体（**01.**、**02.**等）
- 关键字段加粗（**餐饮切入点：**、**营销建议：**）
- 每条新闻之间用分隔线（---）隔开
- 简洁美观，有视觉层次感

**新闻摘要要求：**
- 100-150字左右
- **聚焦餐饮相关内容**，删除与餐饮无关的细节
- 讲清楚事件的核心（谁、什么事、为什么重要）
- 补充有意思的细节（具体数据、关键人物、有趣情节）
- 让读者不看原文就明白为什么引起热议或关注
- 避免空泛的总结，要有具体信息

**示例对比：**

❌ **不好的摘要**（包含无关信息）：
> 特朗普访华晚宴菜单曝光，国宴选用淮扬菜系。晚宴邀请字节跳动CEO梁汝波等科技大佬出席，但腾讯、阿里高管缺席引发热议。网友关注国宴菜品选择背后的文化含义。

✅ **好的摘要**（聚焦餐饮）：
> 特朗普访华晚宴菜单曝光，国宴选用淮扬菜系，包括松鼠鳜鱼、狮子头等经典菜品。网友热议国宴菜品选择背后的文化含义，高端餐饮品牌借势推出"国宴同款"菜品。淮扬菜系关注度飙升，成为高端餐饮营销新热点。

**示例输出：**
```
🍽️ 今日（5月16日）餐饮热点洞察

✍️ 总结：今日热点中，特朗普访华国宴、黄仁勋北京餐饮选择成为餐饮营销切入点。酒店卫生问题持续发酵，餐饮食品安全话题升温。饭店创意营销获千万关注，建议餐饮品牌借势国宴话题、强化食品安全宣传。

### 5. 行业热点筛选

用户说"餐饮行业热点"、"科技行业热点"、"金融热点"时：

**预设行业：**
- 餐饮、科技、金融、教育、医疗、娱乐、房产、汽车、时尚、农业

**执行：**
```bash
# 加载行业关键词
INDUSTRY="餐饮"
KEYWORDS=$(cat ~/.config/china-hotnews/industries.json | jq -r ".industries.\"$INDUSTRY\" | join(\"|\")")

# 从各平台筛选行业相关热点
curl -s "https://60s.viki.moe/v2/weibo" | jq -r ".data | .[] | select(.title | test(\"$KEYWORDS\"; \"i\")) | \"• [\(.title)](\(.link)) (\(.hot_value)热度)\"" | head -5
```

**自定义行业关键词：**
```bash
# 用户可以自定义关键词筛选
curl -s "https://60s.viki.moe/v2/weibo" | jq -r ".data | .[] | select(.title | test(\"咖啡|奶茶|火锅|烧烤\"; \"i\")) | \"• \(.title)\""
```
